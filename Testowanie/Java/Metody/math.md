# Math

| Metoda | Cel | Przykładowe zastosowanie |
| --- | --- | --- |
| `Math.abs(x)` | Zwraca wartość bezwzględną liczby. | Obliczanie odległości na osi liczbowej. |
| `Math.pow(x, y)` | Podnosi liczbę `x` do potęgi `y`. | Obliczanie potęg, np. w zadaniach matematycznych. |
| `Math.sqrt(x)` | Zwraca pierwiastek kwadratowy `x`. | Obliczanie długości przekątnej w prostokącie. |
| `Math.round(x)` | Zaokrągla liczbę do najbliższej całkowitej. | Wyświetlanie zaokrąglonego wyniku obliczeń. |
| `Math.ceil(x)` | Zwraca najmniejszą liczbę całkowitą większą lub równą `x`. | Wyznaczanie liczby stron w paginacji. |
| `Math.floor(x)` | Zwraca największą liczbę całkowitą mniejszą lub równą `x`. | Obliczanie przedziałów w histogramach. |
| `Math.max(x, y)` | Zwraca większą z dwóch liczb. | Porównywanie wyników w grach czy algorytmach. |
| `Math.min(x, y)` | Zwraca mniejszą z dwóch liczb. | Wyznaczanie minimum w danych statystycznych. |
| `Math.random()` | Generuje losową wartość z przedziału [0, 1). | Losowanie liczb w grach czy symulacjach. |
| `Math.sin(x)` | Oblicza sinus kąta (radiany). | Wyznaczanie współrzędnych punktu na okręgu. |
| `Math.cos(x)` | Oblicza cosinus kąta (radiany). | Symulacje fizyczne, np. ruch jednostajny po okręgu. |
| `Math.log(x)` | Zwraca logarytm naturalny liczby `x`. | Analiza danych w logarytmicznej skali. |
| `Math.exp(x)` | Oblicza e^x. | Modele wzrostu wykładniczego, np. populacja. |
